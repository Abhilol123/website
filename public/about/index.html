About




This website is created by Abhinav as a Full Stack Development project to learn HTML, CSS, JavaScript. It showcases various projects which can be seen in the projects section. It also showcases the creativity and designing skills.

Abhinav Chandrasekar is currently in National Instite of Technology, Karnataka persuing a Batchelors Degree in Mechanical Engineering. Abhinav is very pationate about engineering. He has taken in a lot of interest in new upcoming Technology such as Machine Learning, Artifitial Inteligence etc. His skills include 3D design using various CAD tools, 3D Printing, Machine Learning and Artificial Inteligence, Web Development. He has a lot of projects in ML and Web-Dev posted in his GitHub page for everyone to find.
Abhinav's ultimate goal is to make the world a better place to live using technology and engineering as a tool.
Originally from India, he has travelled around the world and gained a lot of multi-linguistic as well as multi-cultural experiences.
